Vue scoped 原理（概念示意）

[data-v-xxx] 属性选择器 + 编译期自动加属性

我在 data-v-demo 容器内（紫）→ 模拟 scoped 命中

我在容器外（不变，证明 scoped 隔离）

.vue 里写 <style scoped>，编译后变成 .sel-scoped-p[data-v-7a9e1f]，并给组件元素自动加 data-v-7a9e1f，实现样式只作用于本组件。本质是「属性选择器 + 编译期加属性」，class 只是选择器之一。
