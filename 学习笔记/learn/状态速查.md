# CSS 状态（伪类 Pseudo-class）速查表

> 来源：`learn/test03.html` 中实际演示过的所有伪类。伪类用于选中元素在「某种状态/情形」下的样子，多数**不需要 class**，元素进入该状态即自动命中。

## 一、用户交互状态（鼠标 / 键盘 / 焦点）

| 伪类 | 代表的状态 | 典型用途 |
|------|-----------|---------|
| `:link` | 未访问过的链接 | 默认链接色 |
| `:visited` | 已访问过的链接 | 点过的链接变色 |
| `:hover` | 鼠标悬停其上 | 按钮/链接高亮 |
| `:active` | 被按下/激活的瞬间 | 点击下压效果 |
| `:focus` | 获得键盘焦点 | 输入框聚焦加边框 |
| `:focus-within` | 自身或子孙中任一获得焦点 | 表单整组高亮 |
| `:focus-visible` | 焦点且需要被看到（键盘导航时） | 避免鼠标点击也显示焦点框 |

> 链接四态记忆顺序：`:link → :visited → :hover → :active`（LVHA）。

## 二、结构 / 位置状态（在 DOM 树中的位置）

| 伪类 | 代表的状态 |
|------|-----------|
| `:first-child` | 父元素的第一个子元素 |
| `:last-child` | 父元素的最后一个子元素 |
| `:first-of-type` | 同类型标签里的第一个 |
| `:last-of-type` | 同类型标签里的最后一个 |
| `:only-child` | 父元素里唯一的子元素 |
| `:only-of-type` | 同类型里唯一的一个 |
| `:nth-child(n)` | 父元素里第 n 个子元素（`2n`偶数/`2n+1`奇数/`3n`等公式） |
| `:nth-last-child(n)` | 从末尾数第 n 个 |
| `:nth-of-type(n)` | 同类型里第 n 个 |
| `:nth-last-of-type(n)` | 同类型从末尾数第 n 个 |
| `:root` | 文档根元素（等同 `<html>`） |
| `:empty` | 元素内部没有任何子节点/文字 |

> `tr:nth-child(2n)` 是做斑马纹表格/隔行变色的常用手法。

## 三、表单 / 输入状态（由元素属性/值决定）

| 伪类 | 代表的状态 |
|------|-----------|
| `:enabled` | 控件可用（未禁用） |
| `:disabled` | 控件被禁用 |
| `:checked` | 单选/复选框被选中 |
| `:indeterminate` | 复选框处于半选/不确定状态 |
| `:default` | 表单里默认提交的控件 |
| `:required` | 带 required 属性（必填） |
| `:optional` | 非必填 |
| `:valid` | 输入通过校验 |
| `:invalid` | 输入未通过校验 |
| `:in-range` | 数字/日期值在 min~max 范围内 |
| `:out-of-range` | 值超出范围 |
| `:read-only` | 输入框 readonly（只读） |
| `:read-write` | 输入框可写 |
| `:placeholder-shown` | 占位提示文字正在显示（还没输入） |

## 四、逻辑 / 组合伪类（函数型，可嵌套其他选择器）

| 伪类 | 代表的状态 / 作用 |
|------|-----------|
| `:not(X)` | 不匹配 X 的元素（取反） |
| `:is(X, Y)` | 匹配 X 或 Y（分组简写，权重取最大） |
| `:where(X, Y)` | 同 `:is` 但权重永远为 0 |
| `:has(X)` | 内部包含 X 时命中（父选子，CSS 新特性） |

## 五、其他状态

| 伪类 | 代表的状态 |
|------|-----------|
| `:target` | 元素是当前 URL 锚点（#id）指向的目标 |
| `:lang(xx)` | 元素 lang 属性匹配某语言 |
| `:fullscreen` | 元素处于全屏模式 |
| `:playing` / `:paused` | 音视频正在播放 / 暂停 |

## 一句话记忆

- **交互状态**：`:hover` `:active` `:focus` `:link` `:visited` —— 用户动起来才变；
- **位置状态**：`:first-child` `:nth-child` `:empty` `:root` —— DOM 里的排行；
- **表单状态**：`:checked` `:disabled` `:valid` `:required` —— 控件自身决定；
- **逻辑状态**：`:not()` `:is()` `:has()` —— 用函数组合条件。
