# HTML 属性速查表（Attribute Reference）

> 来源：`test01.html` 中实际演示过的所有**原生 HTML 属性**（不含 Vue 专属）。每个属性标注「作用」与「常用所在标签」。

## 一、全局属性（任何标签都能用）

| 属性 | 作用 | 常用标签 |
|------|------|---------|
| `class` | 指定 CSS 类名，用于挂样式 / 选中元素 | 全部 |
| `id` | 元素唯一身份证（页面内不重复） | 全部 |
| `style` | 行内样式，直接写 CSS | 全部 |
| `title` | 鼠标悬停提示气泡 + 无障碍信息 | 全部 |
| `lang` | 元素内文字语言（如 `zh-CN`） | `<html>` 等 |
| `dir` | 文字方向 `ltr`/`rtl` | 全部 |
| `hidden` | 存在即隐藏元素 | 全部 |
| `tabindex` | 键盘 Tab 键聚焦顺序 | 全部 |
| `draggable` | 是否可拖拽 `true`/`false` | 全部 |
| `contenteditable` | 是否可直接编辑文字 `true`/`false` | 全部 |
| `spellcheck` | 是否开启拼写检查 `true`/`false` | 输入类 |
| `translate` | 是否允许翻译工具翻译 `yes`/`no` | 全部 |
| `accesskey` | 指定键盘快捷键 | 全部 |
| `data-*` | 自定义数据属性（JS 用 `dataset` 读） | 全部 |
| `role` | ARIA 角色，告诉读屏软件元素身份 | 全部 |
| `aria-label` | 无障碍：元素的可读名称 | 全部 |
| `aria-hidden` | 无障碍：是否对读屏隐藏 | 全部 |
| `aria-expanded` | 无障碍：折叠/展开状态 | 可折叠元素 |
| `itemscope` | 微数据作用域（配合 itemprop 标注结构化数据） | 全部 |
| `itemprop` | 微数据：给元素标注业务字段名 | 全部 |

## 二、链接 / 媒体相关

| 属性 | 作用 | 常用标签 |
|------|------|---------|
| `href` | 链接目标地址 | `<a>` `<link>` |
| `target` | 打开方式 `_blank`/`_self` | `<a>` |
| `rel` | 与目标页关系说明（如 `noopener`） | `<a>` `<link>` |
| `download` | 提示浏览器下载而非打开 | `<a>` |
| `src` | 资源地址（图/视频/音频/iframe/script/link） | `<img>` `<video>` `<audio>` `<iframe>` `<script>` `<link>` |
| `alt` | 图片替代文本（加载失败/读屏显示） | `<img>` |
| `width` / `height` | 宽 / 高（像素） | `<img>` `<video>` `<canvas>` |
| `loading` | 图片懒加载 `lazy`/`eager` | `<img>` |
| `controls` | 视频/音频显示播放控件 | `<video>` `<audio>` |
| `autoplay` | 自动播放 | `<video>` `<audio>` |
| `muted` | 静音 | `<video>` |
| `poster` | 视频封面图 | `<video>` |
| `loop` | 循环播放 | `<video>` `<audio>` |

## 三、表单 / 输入相关

| 属性 | 作用 | 常用标签 |
|------|------|---------|
| `type` | 输入类型（text/password/email/number/checkbox/radio/file/date…） | `<input>` |
| `name` | 字段名，表单提交时作 key | `<input>` `<select>` `<textarea>` |
| `value` | 默认值 / 当前值 | `<input>` `<textarea>` `<option>` |
| `placeholder` | 输入框灰字提示 | `<input>` `<textarea>` |
| `required` | 必填，不填无法提交 | `<input>` |
| `disabled` | 禁用（变灰不可操作） | `<input>` `<button>` `<select>` |
| `readonly` | 只读（可见不可改） | `<input>` `<textarea>` |
| `checked` | 单选/复选框默认选中 | `<input type=checkbox/radio>` |
| `selected` | 下拉项默认选中 | `<option>` |
| `maxlength` / `minlength` | 最大 / 最小输入长度 | `<input>` `<textarea>` |
| `min` / `max` / `step` | 数字/日期范围与步长 | `<input type=number>` |
| `pattern` | 正则校验格式 | `<input>` |
| `autocomplete` | 是否启用浏览器自动填充 `on`/`off` | `<input>` `<form>` |
| `for` | label 绑定输入框 id（点文字可聚焦） | `<label>` |
| `multiple` | 允许选多个文件 / 多项 | `<input type=file>` `<select>` |
| `accept` | 限制上传文件类型（如 `image/*`） | `<input type=file>` |

## 四、表格相关

| 属性 | 作用 | 常用标签 |
|------|------|---------|
| `colspan` | 单元格横向跨几列 | `<td>` `<th>` |
| `rowspan` | 单元格纵向跨几行 | `<td>` `<th>` |
| `scope` | 表头 `th` 声明行头/列头 `row`/`col` | `<th>` |
| `border` | 表格边框（旧属性，现多用 CSS） | `<table>` |
| `cellpadding` | 单元格内边距（旧属性） | `<table>` |
| `cellspacing` | 单元格间距（旧属性） | `<table>` |

## 五、结构 / 头部相关

| 属性 | 作用 | 常用标签 |
|------|------|---------|
| `charset` | 文档字符编码（如 `utf-8`） | `<meta>` |
| `name="viewport"` | 移动端视口配置 | `<meta>` |
| `sandbox` | 给 iframe 加安全限制 | `<iframe>` |
| `defer` | 脚本延迟到 DOM 解析完再执行 | `<script>` |
| `async` | 脚本异步加载不阻塞页面 | `<script>` |
