# HTML 标签速查表（Tag Reference）

> 来源：`test02.html` 中实际演示过的所有标签。每个标签标注「作用」，并注明是否自闭合、语义类型。

## 一、文档结构类

| 标签 | 作用 | 备注 |
|------|------|------|
| `<!DOCTYPE html>` | 声明文档为 HTML5 | 不是标签，是声明 |
| `<html>` | 页面根元素 | 包裹整个文档 |
| `<head>` | 文档头部，放元信息（不显示） | |
| `<body>` | 页面主体，放可见内容 | |
| `<title>` | 浏览器标签页标题 | 在 head 内 |
| `<meta>` | 元信息（编码/视口/SEO） | 自闭合 |
| `<link>` | 引入外部资源（多为 CSS） | 自闭合 |
| `<style>` | 页面内嵌 CSS | |
| `<script>` | 引入/编写 JS | |
| `<base>` | 设定相对链接基准地址 | 自闭合 |
| `<noscript>` | 浏览器禁用 JS 时显示的内容 | |

| 标签 | 作用 | 备注 |
|------|------|------|
| `<header>` | 页头 / 区块头部 | |
| `<footer>` | 页脚 / 区块底部 | |
| `<nav>` | 导航区 | |
| `<main>` | 页面主体（应唯一） | |
| `<aside>` | 侧边栏 / 附属内容 | |
| `<section>` | 有主题的内容分区 | |
| `<article>` | 独立完整的内容（文章/评论） | |
| `<figure>` | 图文媒体块 | |
| `<figcaption>` | figure 的标题说明 | |
| `<details>` | 可展开/收起的详情块 | |
| `<summary>` | details 的可点击标题 | |
| `<dialog>` | 对话框 / 弹窗 | |
| `<address>` | 联系方式信息 | |
| `<hgroup>` | 标题分组 | |
